Docker 
  1. create a spring boot App
  2. create docker file "Dockerfile"
  3. create jar spring boot app
  4 .open sts  terminal
  5. build to docker image write command  "Docker build . -t my app" or "Docker build -t my app ." or  Docker build -f Dockerfile2. -t my app
  6. check in docker cmd docker image
  7.docker run -p 8787:8787 my app
